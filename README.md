# Selenium WebDriver with Python

This repository contains Selenium WebDriver practice programs developed while learning automation testing using Python.

## Topics Covered

- Selenium WebDriver Setup
- Different Types of Locators
- Handling Web Elements
- Browser Commands
- Navigation Commands
- Waits (Implicit & Explicit)
- Alerts
- Frames
- Window Handling
- Mouse Actions (ActionChains)
- JavaScript Executor
- Screenshots

## Technologies Used

- Python
- Selenium WebDriver
- ChromeDriver
- PyCharm

## Purpose

This repository showcases my hands-on practice in Selenium WebDriver and serves as a learning resource for web automation testing.
